## To Do Project

A simple to-do list app with Motoko language.


## Requirements

### Environment

- Ensure that your dfx version is set to `0.16.1`:

```bash
dfx --version
```

### install node_module

```bash
npm install
```

### Change Directory

```bash
cd motoko_workshop
```

### Run

```bash
dfx start --clean
```

- Launch the deploy in another terminal:

```bash
cd src/motoko_tutorial_backend/
```

```bash
dfx deploy
```
